Microtask 134 — Add pulse conversion method
File modified: typeclasses/characters.py

Add:

def process_learning_pulse(self):
    for skill_name, data in self.db.skills.items():
        if data["mindstate"] <= 0:
            continue

        # convert some mindstate → rank
        gain = 1
        data["rank"] += gain

        # reduce mindstate
        data["mindstate"] = max(0, data["mindstate"] - 5)

⚠️ intentionally simple for now
