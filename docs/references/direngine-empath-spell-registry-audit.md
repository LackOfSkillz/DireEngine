## Summary

This dispatch audited every registry row whose `allowed_professions` contains `empath` against the canonical Empath wound-transfer mechanic.

Classification summary:

- Canon-compliant: 1
- Canon-violating: 1
- Grandfathered exceptions: 13

Controlling result:

- `regenerate` is the only live Empath-tagged row that currently matches canonical Empath spell behavior closely enough to keep `gsl_2004`.
- `empath_heal` is canon-violating in its current structured form because it can target another character and directly restore HP through the shared healing handler instead of routing through wound transfer.
- The remaining Empath-accessible rows are not canonical Empath class-book coverage. They stay live as grandfathered shared fundamentals or approved `magic_3_0_design` exceptions.

   - proves `empath_heal` currently succeeds on another character
   - proves the target receives direct HP restoration
