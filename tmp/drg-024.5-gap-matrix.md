# DRG-024.5 Gap Matrix

Read-only scope matrix derived from the DRG-024.5 pre-flight.

Purpose:
- Identify which existing magic components are safe to preserve for DRG-024.5a.
- Isolate the smallest rewrite surface required for canon reconciliation.
- Separate starter spell content work into DRG-024.5b.
- Separate advanced magic systems into DRG-024.5c.

Classification rules:
- `Keep`: matches canon closely enough; no DRG-024.5a implementation change required.
- `Rewrite`: exists, but DRG-024.5a must change it because the current behavior diverges from canon.
- `Add`: missing infrastructure that DRG-024.5a needs to introduce.
- `Defer`: exists or is known-missing, but belongs to DRG-024.5b or DRG-024.5c rather than DRG-024.5a.

## Narrow DRG-024.5a Rewrite Set

If DRG-024.5a is kept disciplined, the primary implementation surface should be:

- `engine/services/mana_service.py`
- `typeclasses/characters.py`
- `commands/cmd_release.py`
- `commands/default_cmdsets.py`
- new `commands/cmd_harness.py`
- targeted lifecycle tests in `tests/services/test_mana_service.py`
- targeted runtime tests in `tests/services/test_character_spell_runtime.py`

Everything else should default to preserve-in-place unless the lifecycle rewrite proves otherwise.

## Release Command Preservation Note

`release` should not drop empath-link functionality.

Recommended DRG-024.5a direction:
- `release` with no qualifier should prefer canonical magic release behavior when the caller has a prepared spell, held mana, or active cyclic spell.
- `release link` should preserve the current empath-link behavior explicitly.
- `release spell`, `release mana`, and bare `release` should converge on the canonical DR spell-release family.

This keeps the command canonical for magic without breaking the empath workflow already present in the repo.

## Drafting Implication

Use this matrix as the scope fence for DRG-024.5a:

- Rewrite only the loop owners.
- Add only the missing canonical verb and timing seams.
- Do not broaden into catalog, slots, feats, cambrinth, or ritual systems.
- Treat cyclic work as seam cleanup only unless the rewrite reveals a hard blocker.
