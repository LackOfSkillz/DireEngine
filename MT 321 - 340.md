accuracy curve
damage profile
roundtime
optimal usage context

This is not cosmetic—it defines playstyle

2. Multi-damage model (critical to DR feel)

Weapons distribute damage across:

Impact
Slice
Puncture

Armor interacts differently with each (you’ll hook this later).

👉 This is why maces feel different from swords.

3. Balance is a first-class stat

Balance affects:

hit chance
consistency

High balance:

reliable hits
Low balance:
swingy, harder to use
4. Suitability (skill-to-weapon relationship)

Weapons don’t just scale linearly with skill.

Low skill:

weapon feels awkward
lower effectiveness

High skill:

weapon “comes alive”

👉 This is exactly aligned with your unlock system idea

5. Roundtime is weapon-driven pacing

Weapons define tempo:

daggers → fast, low damage
greatswords → slow, heavy

This creates:

tempo identity

6. Hit quality matters (very important)

DR doesn’t just do hit/miss.

It evaluates:

glancing
solid
crushing

Which affects:

damage
messaging

You already started this—good.

7. Weapon verbs reinforce identity
slice weapons → slash/cut
blunt → crush/smash
puncture → stab/pierce

This matters psychologically.

8. Skill affects BOTH:
accuracy
damage effectiveness

Not just one.

9. No hard gating

Anyone can use anything.

But:

low skill = bad performance
high skill = unlock potential

def get_weapon_effects(self, character):
    effects = {}
    skill_name = self.db.weapon_profile.get("skill", "brawling")
    rank = character.get_skill(skill_name)

    for tier in self.db.skill_scaling.get(skill_name, []):
        if rank >= tier["rank"]:
            effects.update(tier["effects"])

    return effects
