DragonRealms Weapon System (what actually matters)
1. Weapons are skill-based, not level-based
Same as armor
Anyone can use anything
Skill determines effectiveness
2. Weapon skill categories (core identity)

DR uses distinct weapon skills:

Melee:
Small Edged (daggers)
Large Edged (broadswords)
Two-Handed Edged
Small Blunt
Large Blunt
Two-Handed Blunt
Polearms
Brawling
Ranged:
Short Bow
Long Bow
Crossbow
Slings / Thrown (less relevant early)

👉 These are not cosmetic—they define:

accuracy
damage effectiveness
learning
3. Damage is multi-dimensional (critical)

Weapons don’t just do “damage”.

They have:

Impact
Slice
Puncture

Each weapon has a profile mix, for example:

sword → slice dominant
mace → impact dominant
spear → puncture dominant
4. Balance + Suitability (VERY important)

Each weapon has:

Balance → accuracy / ease of use
Suitability → how well it fits the user

Low skill = weapon feels “off”
High skill = weapon becomes more effective

5. Roundtime (RT) is weapon-driven
Heavier weapons = longer RT
Faster weapons = lower damage
6. Damage is NOT flat

Damage depends on:

weapon base
skill
hit quality (glancing / solid / etc.)
armor interaction
7. Attack verbs + messaging matter

Different weapons produce different messaging:

slash
stab
crush
lunge

This is not fluff—it reinforces weapon identity.

8. Position + stance matter (you already built this)
certain weapons perform better in certain conditions
9. Learning is tied to difficulty
trivial fights = no learning
too hard = reduced learning

def get_weapon_effects(self, char):
    effects = {}
    rank = char.get_skill(self.db.skill)

    for tier in self.db.skill_scaling.get(self.db.skill, []):
        if rank >= tier["rank"]:
            effects.update(tier["effects"])

    return effects
