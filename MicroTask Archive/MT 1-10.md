Microtask 9 — Create Stats Command
Objective

Expose character state in-game.

Instructions

Create file:

commands/cmd_stats.py

Add:

from evennia import Command

class CmdStats(Command):
    key = "stats"

    def func(self):
        char = self.caller

        hp, max_hp = char.get_hp()
        target = char.db.target.key if char.db.target else "None"
        combat = char.db.in_combat

        char.msg(f"HP: {hp}/{max_hp}")
        char.msg(f"Target: {target}")
        char.msg(f"In Combat: {combat}")
Constraints
No colors
No formatting

Confirm no import errors
